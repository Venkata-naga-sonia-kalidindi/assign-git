•Why should you have different branches?
•In a project how many remote Git repositories do you usually have?

-gitignore file is used to ignore the purposefully untracked files while commiting. A gitignore file in java project contains *.war, *.class,*.jar, build/, bin/ etc.

-Fastforward in git means to forward the branch controls from source to the latest commit,without creating a redundant commit.

-Branching lets user to isolate and develop a feature without getting interrupted by others work.

-several repositories can be used for a project , but only one repository can have "origin".
